# coursera-react-basics-customizing-project
Project: Learning to customize a basic out of the box react application.
# Setting up a React project in VS Code
If you have _NPM_ setup on your computer (or if you're doing CodeSpace here) then you can proceed with the command.
Run command `npm init react-app [appNameHere]` to start a new app
Change Directory into your [appNameHere] and `npm start` will start up build your app and most likely will give you a option to open it up in a browser to view with a port of 3000.
